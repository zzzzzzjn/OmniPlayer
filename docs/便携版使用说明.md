# OmniPlayer Portable

双击根目录的 `OmniPlayer.exe` 即可使用，不需要另外寻找 mpv、Studio、orender 或 bridge。

## 第一次运行

1. 打开“组件与路径”，确认路径均指向当前包内的 `runtime` 目录。
2. 点击“保存设置并修复 bridge_path”。原配置会先创建时间戳备份。
3. 回到“播放控制”，选择音频输出、拖入片源并启动。

## Dante 7.1.4

- 先启动 Dante Virtual Soundcard，建议设置为 ASIO、16×16、48 kHz。
- 在 OmniPlayer 的“音频输出”选择 `Dante Virtual Soundcard (x64) [ASIO]`。
- Windows DVS Tx 1–12 对应当前 7.1.4 的十二路渲染输出；在 Dante Controller 中订阅到接收端 Rx 1–12。

## 程序结构

`OmniPlayer.exe` 是统一入口。mpv 与 Studio 仍是各自的受控子进程，这是它们稳定工作的正式架构，不是依赖外部安装。第三方授权与对应源码地址见 `legal` 目录。
