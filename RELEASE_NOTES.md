# OmniPlayer v1.3

Initial public release of the Windows integration launcher.

## Highlights

- One UI for mpv-omniphony, Omniphony Studio, orender and a compatible bridge
- Portable runtime layout with automatic relative-path discovery
- Black/purple interface and embedded multi-resolution OmniPlayer icon
- Safe YAML bridge-path editing with timestamped backups
- OSC supervision with the in-player object overlay disabled by default
- ASIO/WASAPI enumeration and Dante Virtual Soundcard selection
- Chinese UI, diagnostic logging and drag-and-drop media launch

## Positioning

OmniPlayer is an integration, configuration and launch utility. It does not implement or claim authorship of proprietary codec technology. Playback, rendering, bridge and codec functionality belongs to the separately licensed upstream projects credited in the README and `legal/` directory.
